# Police Pumpkind Custard

## Installation

Pour que la police Pumpkind Custard fonctionne correctement, vous devez:

1. **Télécharger la police Pumpkind Custard** (fichier .ttf)
   - Recherchez "Pumpkind Custard font" sur Google
   - Téléchargez le fichier .ttf depuis un site de polices gratuit

2. **Placer le fichier dans ce dossier**
   - Renommez le fichier en `PumpkindCustard.ttf`
   - Placez-le dans: `/assets/fonts/PumpkindCustard.ttf`

3. **Redémarrer l'application**
   - Si vous utilisez Expo Go, fermez et rouvrez l'app
   - Si sur web, rafraîchissez le navigateur

## Police de fallback

Si la police Pumpkind Custard n'est pas trouvée, l'application utilisera la police système par défaut.
