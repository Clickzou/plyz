# Plyz — Google Play (Japanese)

## Description courte (max 80 caractères)
お気に入りのスターと、デジタルデジテーションやサイン、プライベートビデオ通話を予約しよう。

## Description complète
Plyzはファンと彼らのお気に入りのアーティスト、シンガー、アスリート、クリエイター、クラブを繋ぐプラットフォームで、パーソナライズされたサービスを予約できます。ライブプライベートビデオ通話を予約したり、署名入りのデジテーション・サインを注文したり、ライブイベントに参加したり—すべてがシンプルで安全、ユーザーフレンドリーなアプリの中で可能です。

★ ファンの皆へ

• 検証済みセレブリティの壁を発見
• あなたの写真にパーソナライズされたサインをリクエスト
• スターとのプライベートビデオ通話を予約
• QRコードまたはイベントコードでライブデジテーションイベントに参加
• 50以上のフォントの中から、手書き署名やデジテーションで写真をカスタマイズ
• あなたのギャラリーで全てのデジテーションと最高の思い出を見つける
• 100%安全な支払い

★ 舞台裏に没入しよう

• 舞台裏の様子をあなたに見せてくれるパーソナリティとのライブセッションを予約
• コンサート：リハーサル、ラウンジ、バックステージ、ステージ登場
• スポーツ：ロッカールーム、ウォームアップ、トレーニング、試合前
• まるでそこにいるかのようにあなたのアーティスト、クラブ、チームをフォロー

★ セレブリティ、クリエイター & クラブの皆へ

• セレブリティモードを数秒で有効化
• ファンにデジテーション、サイン、ビデオ通話を提供
• QRコード付きの独自ライブイベントを作成
• あなたの価格を設定して、支払いを直接受け取る
• ニュースを公開してコミュニティとの連絡を保つ
• あなた専用スペースから収益を追跡

★ シンプル、安全、配慮ある

• 検証済みセレブリティアカウント
• Stripeで処理された支払い — Plyzはあなたの銀行情報を記録しません
• 暗号化されたビデオ通話
• あなたのデータは保護され、決して転売されません
