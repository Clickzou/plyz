## Description courte (max 80 caractères)

## Description complète
Plyzは、お気に入りのアーティスト、シンガー、アスリート、クリエイター、クラブとファンを近づけるアプリです。あなただけのメッセージ、直筆サイン、プライベートなビデオ通話をリクエストしたり、舞台裏をのぞいたり——すべてがシンプルで安全、そして使いやすいアプリの中に。

★ ファンのみなさんへ
• 認証済みセレブが集まる特設フィードをチェック
• あなたの写真に、特別な直筆サインをリクエスト
• スターと1対1のプライベートなビデオ通話を予約
• QRコードやイベントコードで、ライブのサイン会に参加
• 50種類以上のフォントから、手書き風サインやメッセージで写真をデコレーション
• 受け取ったサインや大切な思い出を、すべてギャラリーに保存
• 100%安全な決済

★ 舞台裏をのぞこう
• 本物のファンだけが見られる、特別な瞬間を生配信で体験
• ライブ：リハーサル、楽屋、バックステージ、ステージへの登場
• スポーツ：ロッカールーム、ウォームアップ、練習、試合前の様子
• まるでその場にいるように、推しのアーティストやクラブ、チームを応援

★ セレブ・クリエイター・クラブのみなさんへ
• 数秒でセレブモードをオンに
• ファンにサイン、メッセージ、ビデオ通話を提供
• QRコード付きの自分だけのライブイベントを開催
• 料金を自由に設定し、報酬を直接受け取り
• 最新情報を発信して、コミュニティとつながり続けよう
• 専用ダッシュボードで収益をいつでも確認

★ シンプル・安全・リスペクト
• 認証済みのセレブアカウント
• 決済はStripeが処理——Plyzがあなたの銀行情報を保存することは一切ありません
• 暗号化されたビデオ通話
• あなたのデータは保護され、第三者に売られることはありません

アーティスト、アスリート、クラブ、クリエイター、有名人——どんな推しがいても、Plyzなら、あなたにとって大切な人と、唯一無二の本物の瞬間を過ごせます。
